ACERCA DEL PROYECTO

Autor: Daniel Ramon Campos
Fecha de creación: Enero 2018
Descripción: El juego consiste en frenar la caída de una nave para que esta pueda aterrizar correctamente.

HERRAMIENTAS DE DESARROLLO

El proyecto ha sido desarrollado mediante código html, css, y javaScript.
El diseño del proyecto tiene dos versiones: una versión pensada para pantallas de ordenador horizontales, y una versión optimizada para smartphone

El juego ha sido diseñado consultando los siguientes enlaces:


  • Para el tratamiento del código html y css: http://www.w3schools.com
  • Para reducir el peso de las imágenes: https://tinypng.com/
  • Para editar las imágenes con el programa PHOTOSHOP: http://www.adobe.com/es/products/photoshop.html
  • Para controlar y organizar el estado del proyecto mi cuenta en Github: https://https://github.com/Rikuoh
